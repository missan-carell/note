3. **轴心点介绍：**

   - 质心点：单个物体则是他的原点，若有多个物体则是他们的重心
   - 3d游标：即游标所在位置
   - 外边框中心：即将多个物体拿最小的框框住之后，外边框的中心
   - 各自原点:即选中多个物体后，旋转时或移动还是按照各自原点移动
   - 活动元素：在多选时最后选中的物体就是活动元素。改变活动元素：在多选时按住shift+要更新的物体即可改变

   ![image-20251126182708671](./.assets/image-20251126182708671.png)

4. **坐标介绍：**

   在使用全局/局部坐标的时候，例如G+z，一开始会按照当前坐标系来移动，这时候再次按下方向键（如z），就可以切换坐标方式

   - 全局坐标：即世界坐标，上下左右根据xyz轴来定义，是不变的
   - 局部坐标：即物体本身自带的坐标系，会跟着物体的旋转改变



![image-20251126183428574](./.assets/image-20251126183428574.png)

5. 保存和输出：在右下角的菜单里，可以设置输出的分辨率和文件位置，记得设置合适的位置

### 1.2.1 ==游标和原点==:

1. 游标，即视图中间的红白圆形：

![image-20251126173825866](./.assets/image-20251126173825866.png)

- 游标的功能：

  1）游标在哪里你新建的物体，就会出现在哪里

  2）游标可以作为轴心点来旋转：

  选择在菜单上方的第二个功能，默认为质心点，将其改成3D游标，物体旋转时便会随着游标的位置来旋转

  ![image-20251126175522733](./.assets/image-20251126175522733.png)

- 游标的操作：

  1）在边缘菜单选择游标图形，即可移动游标位置。或 shift + 右键

  ![image-20251126174154463](./.assets/image-20251126174154463.png)

  2）选中物体，shift+s调出排菜单，选择 `选中项-游标`可以将该物体快速移动到游标位置

  3）shift + c ：将游标移动到原点处



2. 原点：即每个物体中心的黄点，物体在进行缩放等操作时会围绕原定进行。计算机只认识原点

   <img src="./.assets/image-20251126175934802.png" alt="image-20251126175934802" style="zoom:50%;" />

  1）修改原点位置：右上角选项，选择仅影响-原点，切换到正视图（前视图），此时按住G会移动的只有原点了，而非模型，可以调整原点的位置。移动完成后注意把仅影响原点关掉

  ![image-20251126180519227](./.assets/image-20251126180519227.png)

### 1.3.2 渲染入门

在blender 里自带了两个渲染，EEVEE，以及CYCLES，两个渲染的功能对比：

| 渲染器 | 优点                                          | 缺点                                                         |
| ------ | --------------------------------------------- | ------------------------------------------------------------ |
| EEVEE  | 快，相对不占资源                              | 没有光线追踪，很多光线阴影的细节不佳，需要额外更多参数设置辅助。 |
| CYCLES | 逼真,高级能模拟出物理世界里非常细节的光线阴影 | 慢,相对更消耗硬件资源                                        |

（表格来源：KurTips）

1. 渲染器的选择：

   在菜单的右下方菜单里可找到渲染器，在确认完渲染器之后通过f12可以快捷渲染

   - 强度：即默认的亮度，越低越暗

# 2. 开始建模

## 2.1 点线面

在开始之前，首先要进入编辑模式，通过Tab可以在物体模式和编辑模式之间切换。但此时切换其他模式便不足够方便，通过这个设置可以解决这个问题：

编辑 -- 偏好设置 -- 键位映射 -- 勾选 <u>拖动激活饼菜单</u>后，回到主视图，按Tab依旧是切换编辑模式，但是长按Tab加移动鼠标，可快捷调出饼菜单

<img src="./.assets/image-20251130132750067.png" alt="image-20251130132750067" style="zoom: 50%;" />

![](./.assets/image-20251130133457470.png)

1. 选择快捷键：

- 按住shift+1/2/3 可以一次性多选点线面

| 快捷键                    | 功能                 | 介绍                                                         |
| :------------------------ | -------------------- | ------------------------------------------------------------ |
| B                         | 框选                 | 画框选中框内物体                                             |
| Ctrl+I                    | 反选                 | 选中之后用此快捷键可以反选                                   |
| C                         | 刷选                 | 用鼠标涂抹选择，鼠标中键擦除选取，滚轮调整大小。esc/点击空白处退出刷选 |
| Shift+选择                | 多选                 | 多选点/线/面                                                 |
| Ctrl+选择点               | 连接点               | 自动连接点之间的最短路径（若无法连接可能是因为刷选导致的）   |
| 选中+L                    | 选中相连点           | 全选和选择目标的所有物体（或点线面），使用时要保持鼠标在选中物体上，否则会无法选中 |
| （物体模式）Shift多选+Tab | 多个物体切换编辑模式 | 可以同时编辑多个物体                                         |
| 选中点/线/面+Alt+双击鼠标 | 选中相连点           | 只会选中相连的一圈点。圈的位置会根据鼠标点击的位置决定，如想纵向选择，可以将鼠标放在选中点的上方 |
| Alt+Z                     | 切换透视模式         | 透视模式可以看到物体内部，后部结构，**此时再使用框选，就可以选到物体的后部** |
