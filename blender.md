## 1.1 操作和快捷键

1. 基础操作：

| 操作                           | 功能                               |
| ------------------------------ | ---------------------------------- |
| 鼠标中键                       | 移动视角，观察3d空间               |
| shift+鼠标中键                 | 平移3d空间                         |
| 鼠标中键滑轮                   | 缩放空间                           |
| Delete                         | 删除选中的模型                     |
| shift+A                        | 添加菜单                           |
| shift+A，在菜单中选择网格      | 新建物体（新物体默认出现在游标处） |
| ctrl+A                         | 应用菜单                           |
| 鼠标右键                       | 取消                               |
| 鼠标左键                       | 确认                               |
| 左键框选（需要在工具栏选框选） | 快速选择多个物体                   |
| Alt+鼠标中键旋转               | 旋转到上下左右的正视图             |

2. 快捷键：最上面为常用的基础功能，后面的快捷键根据需要选择记忆

   注：<u>以下快捷键都需要选中物体后使用</u>

| 快捷键        | 功能                         | 应用场景                                                     |
| :------------ | ---------------------------- | ------------------------------------------------------------ |
| G             | 移动物体                     | 调整外壳位置                                                 |
| R             | 旋转物体                     | 改变物体角度                                                 |
| S             | 缩放物体                     | 调整大小                                                     |
| Tab           | 物体/编辑模式切换            | 进入编辑模型细节                                             |
| Ctrl+R        | 环切（添加一圈线）           | 为模型增加细节或支撑边                                       |
| E（编辑模式） | 挤出                         | 从面或边拉伸出新的结构                                       |
| I（编辑模式） | 内插面                       | 在现有面上插入一个缩小的面，用于开孔                         |
| G/S/R + x/y/z | 在坐标轴上移动/旋转/缩放物体 | 显示坐标轴，更快定位                                         |
| Alt+S         | 撤销缩放                     | 回到原始的大小，而非上一步的大小                             |
| Alt+R         | 撤销旋转                     | 回到原始的角度，而非上一步的角度                             |
| Alt+G         | 撤销移动                     | 回到原始的位置，而非上一步的位置                             |
| H             | 隐藏物体                     | 关闭物体所在图层                                             |
| shift+H       | 把没有选中的物体隐藏         | 关闭没有选中的物体所在图层                                   |
| Alt+H         | 显示所有物体                 | 打开所有图层                                                 |
| shift+D       | 复制+移动物体                | 先复制，然后可以把复制的物体移动出来，右键只能取消移动，不能取消复制 |
| C             | 刷选                         | 用鼠标多选物体                                               |

注意：E（挤出）I（内插面）都需要在编辑模式选中物体的点/线/面后才能使用



- 小键盘专用快捷键：部分台式机或自己配的小键盘（即数字键盘）上的专用快捷键

  若没有小键盘，也可以**点击`~`切出的菜单来快速切换视图**

| 快捷键  | 功能                                |
| ------- | ----------------------------------- |
| 1       | 正视图                              |
| 3       | 右视图                              |
| 7       | 顶视图                              |
| 9       | 翻转当前视图                        |
| 2,4,6,8 | 在当前视图的基础上微调，每次调整15° |
| 5       | 切换正焦视图和透视视图              |
| 0       | 切换摄像机视图                      |
| 小数点  | 将选择物体放大到屏幕中间            |



- 编辑模式快捷键：

| 快捷键 | 功能       |
| ------ | ---------- |
| 1      | 选择**点** |
| 2      | 选择**线** |
| 3      | 选择**面** |
